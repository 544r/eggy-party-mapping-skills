# 蛋仔派对 - 条件积木完整列表

## 通用条件

| 名称 | 功能说明 | 参数设置 |
|------|----------|----------|
| 拥有标签 | 判断对象身上是否有标签 | 参数1: 单位/玩家/角色蛋仔/生物/组件等；参数2: 字符串(标签名) |
| 非 | 条件不成立时，视为成立 | 参数1: 布尔值 |
| 或 | 一旦两个条件有一个成立，就视为成立 | 参数1: 布尔值；参数2: 布尔值 |
| 与 | 只有两个条件都成立，才视为成立 | 参数1: 布尔值；参数2: 布尔值 |
| 比较 | 可以比较数值的大小，对象是否相等 | 参数1: 多种类型；参数2: 数值比较符；参数3: 整数 |
| 对象存在 | 用来判断物体或数据对象有没有被销毁 | 参数1: 单位/玩家/组件等 |
| 自定义属性存在 | 不存在的自定义属性会导致逻辑错误 | 参数1: 单位/玩家等；参数2: 字符串(属性名) |
