# 蛋仔派对 - 控制积木完整列表

## 1. 重复执行
- **功能说明**：重复{#0}次
- **参数**：整数（次数）
- **备注**：一次重复的次数如果过大会导致游戏变卡

## 2. 如果/否则
- **功能说明**：如果以下条件成立
- **说明**：通过条件来区分不同的情况下，执行什么样的动作，完成不同的逻辑

## 3. 遍历列表
- **功能说明**：选取{#0}, 并执行动作
- **参数**：列表类型（单位列表、整数列表、坐标点列表、角色蛋仔列表、生物列表等大量类型）
- **说明**：遍历会从列表的第一个元素开始到最后一个元素截止，通过表达式遍历的某某值来获得当前遍历的元素

## 4. 遍历整数
- **功能说明**：从{#0}遍历到{#1}, 每次增加{#2}，执行动作
- **参数**：1. 整数（起始）, 2. 整数（结束）, 3. 整数（增量）
- **说明**：每次遍历都会让遍历的数增加。注意，从1遍历到4，每次增加1，实际上运行过程是：1，2，3

## 5. 遍历表格的行
- **功能说明**：遍历{#0}的行，并执行动作
- **参数**：表格
- **说明**：将表格的行进行遍历，获得所有的行数信息

## 6. 遍历表格的列
- **功能说明**：遍历{#0}的列，并执行动作
- **参数**：表格
- **说明**：将表格的列进行遍历，获得所有的列数信息

## 7. 子触发器
- **功能说明**：子触发器
- **说明**：动态注册一个触发器，可以根据上文中变量的实际情况进行指定的事件监听

## 8. 运行计时器
- **功能说明**：经过{#0}秒执行下列动作，执行{#1}次，是否立刻执行{#2}
- **参数**：1. 定点数（秒数）, 2. 整数（次数）, 3. 布尔值, 4. 回调函数
- **说明**：执行次数为-1时表示无限次数

## 9. 运行帧计时器
- **功能说明**：经过{#0}帧执行下列动作，执行{#1}次，是否立刻执行{#2}
- **参数**：1. 整数（帧数）, 2. 整数（次数）, 3. 布尔值, 4. 回调函数
- **说明**：执行次数为-1时表示无限次数
