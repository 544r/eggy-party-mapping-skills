# 蛋仔派对移动端 - 蛋码系统完整教程

## 一、蛋码入口

**路径：** 更多 → 蛋码

蛋码是蛋仔工坊的图形化编程工具，通过拼接积木编写触发器逻辑应用到地图中。

---

## 二、蛋码界面布局

### 界面主要区域

| 区域 | 位置 | 功能 |
|------|------|------|
| 触发器列表 | 左侧 | 切换全局逻辑或局部逻辑 |
| 运行日志 | 界面某区域 | 显示错误、警告和信息 |
| 积木库 | 右侧 | 分类选取积木，带搜索功能 |
| 工具栏 | 左下角 | 新建触发器组 |

### 积木库功能

- **搜索功能：** 右下角搜索框
- **说明开关：** 选择积木时可打开查看功能讲解
- **拖拽删除：** 拖动积木时右侧变为垃圾桶样式

### 工具栏操作

**新建触发器组：** 左下角【新建触发器组】按钮

**触发器组管理：** 可重命名、禁用、复制、删除触发器或触发器组

### 积木工具栏（长按/点击画布积木时出现）

| 操作 | 功能 |
|------|------|
| 复制 | 将积木及下方积木放入剪贴板 |
| 单条复制 | 仅复制当前积木 |
| 弹出 | 使积木脱离积木串 |
| 替换 | 替换为其他积木 |
| 注释 | 添加标注框 |
| 添加断点 | 暂停运行进行调试 |
| 禁用 | 积木变灰不生效 |
| 删除 | 删除当前积木 |

### 蛋码设置入口

**路径：** 左上角工坊设置 → 编辑设置 → 蛋码菜单

可修改蛋码菜单的触发方式

## 三、触发器ECA结构

### 创建位置差异

| 类型 | 位置 | 说明 |
|------|------|------|
| 全局触发器 | 蛋码界面主区域 | 地图主逻辑，游戏开始时生效 |
| 对象触发器 | 物体预设 → 蛋码编辑 | 只给某个对象发送指令 |

**核心区别：** "全局触发器是地图的主逻辑"，"对象触发器只给某个对象发送指令"

### 变量规则

- **全局变量：** 所有触发器可用
- **局部变量：** 单个触发器的单个触发器组中使用
- **临时变量：** 仅当前触发器生效

### 子触发器

可在动作中创建，注册后独立运行，使用后可销毁，适合动态逻辑。

## 四、蛋码积木分类

### 积木库分类

| 分类 | 说明 |
|------|------|
| 事件 | 拖出后新建触发器并注册监听 |
| 动作 | 事件触发后的结果，可无限添加 |
| 控制 | 控制动作执行流程（循环、条件等） |
| 条件 | 需嵌入控制积木中使用 |
| 取值 | 嵌套在其他积木中获取数据 |

### 取值方式

- 预设
- 输入值
- 表达式
- 变量

## 五、触发器组管理

**路径：** 左下角【新建触发器组】

可分类管理触发器：

- 重命名
- 禁用
- 复制
- 删除
