# 蛋仔派对移动端 - 技能编辑器完整教程

## 一、技能编辑器入口

**路径：** 更多 → 预设编辑 → 左下角【技能库】

技能以预设作为单位，一个预设就是一个技能模版。

---

## 二、技能预设分类

| 类型 | 位置 | 说明 |
|------|------|------|
| 模版技能 | 【基础】页 | 查看技能描述，修改冷却时间、施法范围、特效等 |
| 官方技能 | - | 包含蛋仔决战技和道具技能 |
| 枪械技能 | - | 用于物品装配，配合【选中物品时角色表现】控制射击镜头 |
| 塔防技能 | - | 《艾比保卫战》塔防玩法技能 |
| 自定义技能 | 自定义分类 → 点击+号 | 获得空白技能模版 |

---

## 三、技能获取与使用入口

| 方式 | 路径 |
|------|------|
| 角色预设添加 | 预设编辑 → 角色分类 → 新增技能 → 跳转添加技能界面 |
| 生物添加 | 生物预设 → AI编辑里编辑搜敌行为 |
| 蛋码动态添加 | 使用蛋码动作积木动态添加/移除技能 |
| 物品主动技能 | 物品预设 → 配置主动技能 → 配置对应技能格 |

**技能格说明：**
- 第一个技能格为攻击技能（普攻机制）
- 技能格编号从0开始计数
- 技能格高级设置中可设置技能编号
- 界面编辑中添加对应的技能格控件

---

## 四、技能构成

技能由三部分组成：
- **触发方式**（冷却、充能）
- **表现效果**（动画、特效、音效）
- **实际伤害**（目标筛选、附加效果）

### 基础信息

| 属性 | 说明 |
|------|------|
| 名称/标签/图标 | 技能标识与分类依据 |
| 描述 | 长按技能键时显示 |
| 击败回收 | 死亡后是否回收技能 |
| 类型 | 基础类、道具类、被动类 |
| 等级/子技能 | 成长性与多段技能设置 |
| 自定义属性 | 蛋码中可用的变量 |

### 技能类型

- **基础类**：玩家按键主动触发
- **道具类**：拾取道具获得，含持有动画、施法隐藏、使用后销毁等设置
- **被动类**：达到条件自动释放，支持自定义事件触发

### 施法阶段

- **冷却时间**：由"激活冷却"锚点位置决定
- **施法时间**：持续秒数，1秒=30帧
- **释放方式**：松开施法、按下持续施法、两段式
- **施法限制**：禁止跳跃/滚动等行为

### 施法方式与目标筛选

- 施法范围圈：可视化范围
- 目标筛选：根据组件类型、阵营、标签筛选

### 指示器类型

| 类型 | 用途 |
|------|------|
| 方向目标 | 矩形/扇形/抛物线 |
| 点目标 | 圆形、狙击瞄准 |
| 指定目标 | 锁定具体目标 |
| 无目标 | 自身或周围释放 |

### 时间轴与锚点

- **锚点**：技能日程表，即时锚点（六边形）单点生效，持续锚点（矩形）持续作用
- **锚点类型**：基础功能、表现效果、范围效果、运动相关、生成物
- **自定义锚点**：可编写蛋码实现自定义逻辑

### 公式编辑

支持数学公式关联变量，如"等级越高冷却越短"。可在自定义公式栏新增公式，蛋码中通过【获取公式属性】调用。

## 五、技能编辑器进阶

### 成长型技能

**界面位置：** 技能配置【基础】页 → 【等级】部分

| 选项 | 说明 |
|------|------|
| 【能否升级】开关 | 打开后技能可升级 |
| 初始等级 | 初始等级，范围0~10 |
| 最大等级 | 最大等级，范围0~10 |

**蛋码配合：** 使用<技能升级>、<技能降级>事件配合等级判断编写逻辑

---

### 可学习技能

**界面位置：** 技能预设的等级设置

| 选项 | 说明 |
|------|------|
| 【可学习】开关 | 打开后玩家可主动消耗技能点升级 |
| 学习需求等级 | 设置高阶技能的解锁等级要求 |

**相关蛋码：** <增加角色技能点>动作积木

---

### 多段切换技能

**界面位置：** 技能预设【基础】页面 → 子技能部分

| 参数类别 | 选项 |
|---------|------|
| 切换方式 | 【施法切换】或【计时切换】 |
| 切换规则 | 顺序、完全随机、不重复随机 |
| 切换参数 | 回归计时、切换刷新、切换间隔、切换后冷却 |
| 计时切换特有 | 初始间隔参数 |

---

### 消耗型技能

**界面位置：** 技能预设【施法】界面 → 【充能】列表

| 选项 | 说明 |
|------|------|
| 【技能消耗次数】开关 | 开启后变为消耗型 |
| 充能类型 | 无效/次数不满时充能/次数为0时充能 |
| 最大使用次数 | 充能层数上限 |
| 充能间隔 | 每次充能需要的时间 |
| 充能数量 | 每次充能恢复的次数 |
| 充能初始次数 | 技能开始时的充能层数 |

**相关蛋码：**
- <设置技能是否为充能技能>
- <设置技能最大释放次数>
- <设置技能充能冷却>
- <设置技能释放次数>

---

### 蓄力技能

**界面位置：** 技能预设【表现】界面

| 属性 | 说明 |
|------|------|
| 能否蓄力 | 打开后技能可蓄力 |
| 蓄力阶段时间 | 蓄力阶段时间轴长度 |
| 满蓄力时间 | 蓄力时间上限 |
| 蓄力阶段结束自动施放 | 时间结束后自动释放 |
| 禁用蓄力时方向同步 | 蓄力时角色面朝方向是否跟随变化 |
| 显示蓄力图标 | 开关蓄力闪电图标显示 |

**相关蛋码：** <获取技能蓄力百分比>积木

---

### 普攻技能

**界面位置：** 角色预设界面 → 【生物技能】部分的攻击技能栏

| 参数 | 说明 |
|------|------|
| 指定技能 | 选择作为普攻的技能预设 |
| 技能槽位 | 点击触发的技能格控件编号 |
| 自动攻击 | 范围内出现符合目标时自动施放 |
| 显示冷却 | 关闭后不显示冷却表现 |

**受角色属性影响：** 攻击范围、攻击间隔、攻击速度
