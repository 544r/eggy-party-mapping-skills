# 蛋仔派对移动端 - 组件编辑完整教程

## 一、生物组件

### 创建方法

**路径：** 组件侧边栏 → 【生物】分类 → 拖拽到场景

> 注意：生物组件无法通过缩放工具调整大小，需在高级设置→【物理】→【通用属性】中修改，缩放范围为0.1~10

**数量限制：** 地图内生物上限为100个，动态创建不限制数量

**生物类型：** 蛋仔生物、人形生物、四足动物生物、鱼形等

---

### 基础外观编辑

**路径：** 高级设置 → 生物属性

可设置项：
- **面部表情**（脸型选择后调整）：高级设置 → 生物属性
- **材质修改**：高级设置 → 表现 → 材质参数

---

### 自定义外观

**路径：** 生物属性 → 启用自定义外观开关 → 进入编辑界面

**操作流程：**
1. 选中组件和生物
2. 右侧选择对应部位作为挂点
3. 组件挂接后可调整：颜色、大小、旋转

**关闭本体显示：** 高级设置 → 表现 → 关闭生物模型显示

**胶囊体编辑：**
- 位置：自定义外观界面左下角
- 参数：胶囊体高度、半径、模型高度偏移

---

### 生物动作

**路径：** 高级设置 → 表现页

可设置：移动动作、跳跃动作、滚动动作、待机动作

**动作生成：** 更多 → 动作生成

支持三种方式：视频生成、拼接官方动作、动作创作

---

### 战斗功能

#### 健康属性

**路径：** 生物属性 或 预设

| 属性 | 位置 |
|------|------|
| 是否无敌 | 生物属性/预设 |
| 是否无限重生 | 生物属性/预设 |
| 掉落类型 | 生物属性/预设 |
| 血条样式 | 生物属性/预设 |
| 血条显示 | 生物属性/预设 |
| 命数 | 生物属性/预设 |
| 健康值 | 生物属性/预设 |

#### 战斗属性

**路径：** 高级设置【属性】或预设

| 属性 | 说明 |
|------|------|
| 移动速度 | 范围0~30 |
| 攻击力 | 复杂模式后显示 |
| 护甲 | 复杂模式后显示 |

#### AI编辑（战斗行为）

**路径：** 高级设置底部 或 预设

| 参数 | 说明 |
|------|------|
| 搜敌范围 | 触发搜敌的半径范围 |
| 反应距离 | 触发攻击的距离 |
| 厌恶距离 | 停止追击的距离 |
| 厌恶时间 | 放弃追击的时间 |
| 生效阵营 | 根据阵营关系判断 |
| 生效标签 | 拥有该标签的目标 |
| 忽略标签 | 排除该标签的目标 |
| 移动方式 | 直线运动/避障运动 |

---

### 基础互动

**路径：** 高级设置 → 【物理】→ 互动列表

| 功能 | 设置位置 |
|------|----------|
| 名称显示 | 表现页-生物属性 |
| 标签 | 基础-通用属性 |
| 所属玩家 | 基础-通用属性 |
| 物理碰撞 | 物理-通用属性 |
| 生物质量 | 物理-通用属性（0.1~100） |
| 抓举互动 | 物理-通用属性 |
| 击倒失控 | 物理-通用属性 |
| 镜面反射 | 表现 |
| 模型显示 | 表现 |

---

### 对话功能

**对话编辑器：** 更多 → 剧情对话

**添加对话互动：**
1. 高级设置 → 【物理】→ 互动列表 → 开启互动
2. 新增互动 → 类型选择【剧情对话】
3. 编辑互动内容 → 选择已创建的对话

**关联任务：** 开启后根据玩家任务状态智能显示对话按钮

---

### 任务功能

**任务编辑器：** 更多 → 【任务】

**添加任务互动：**
1. 高级设置 → 【物理】→ 互动列表 → 开启互动
2. 新增互动 → 类型选择【任务接取】
3. 选择已创建的任务

---

### 商店功能

**商店编辑器：** 更多 → 【物品系统】→ 【物品商店】

**添加商店互动：**
1. 高级设置 → 【物理】→ 互动列表 → 开启互动
2. 新增互动 → 类型选择【物品商店】
3. 选择已创建的商店

---

## 二、地形组件

### 创建方法

**路径：** 组件库 → 结构 → 基础形状

每个地图最多10个地形组件，原点位于角落，不支持旋转或镜像。

**最大尺寸：** 254×254

---

### 地形编辑器（画笔工具）

**位置：** 选中地形组件后出现的工具栏

| 画笔类型 | 功能 |
|---------|------|
| 高度笔刷 | 绘制地形高低起伏，生成物理碰撞 |
| 平滑笔刷 | 打磨棱角，减小高度差 |
| 整平笔刷 | 创造平坦地形 |
| 材质笔刷 | 更改地形纹理样式 |
| 染色笔刷 | 叠加局部染色效果 |
| 装饰笔刷 | 绘制植被等装饰物 |
| 流体笔刷 | 绘制流水景观 |

所有画笔支持调节**笔刷大小**和**笔刷力度**

---

### 地形高级设置

**路径：** 组件高级设置

| 选项 | 说明 |
|------|------|
| 整体缩放开关 | 控制是否等比例缩放 |
| 地块长度/宽度 | 修改地形大小（上限254×254） |
| 细节精度 | 调整顶点密度 |
| 精准采样 | 多地形衔接时使边缘更自然 |

**材质设置：** 高级设置 → 表现栏

| 选项 | 说明 |
|------|------|
| 自定义颜色 | 调整颜色 |
| 移动轨迹 | 调整移动方式 |
| 金属度 | 调整金属反射 |
| 自发光强度 | 调整发光效果 |
| 贴花重复度 | 调整贴花密度 |
| 粗糙度范围 | 调整粗糙程度 |
| 纹理旋转 | 旋转纹理方向 |

---

### 植被风场

**位置：** 地形左上角工具栏

| 功能 | 说明 |
|------|------|
| 全局风场 | 影响所有开启风力标志的植被 |
| 固定风场 | 仅影响指定区域（机关→逻辑→固定风力场） |

---

## 三、地图换色

**路径：** 更多 → 地图换色

| 区域 | 功能 |
|------|------|
| 一键配色Tab | 4个主色块+6个辅助色 |
| 色板编辑 | 自定义颜色，可上传参考图片生成 |

**主色：** 替换主要色调（颜色占比从左至右递减）
**辅助色：** 替换点缀色调

---

## 四、碰撞预览

**路径：** 更多 → 碰撞预览

**预览模式切换：** 界面上方

| 模式 | 说明 |
|------|------|
| 线框模式 | 渲染没有物理的组件，物理体积以线框形式显示 |
| 实体模式 | 物理体积以实体形式渲染 |
| 动态物理模式 | 渲染没有物理的组件，识别动态物理 |
| 运动器表现器模式 | 渲染所有组件，识别运动器/表现器 |
| 遮挡镜头模式 | 渲染所有组件，识别"遮挡玩家时规则"状态 |

**颜色说明：** 左下角查看颜色对应状态

**试玩时预览：** 右上角性能面板 → 碰撞预览按钮

**注意：** 运行时的线框预览仅支持一定范围内的线框预览
