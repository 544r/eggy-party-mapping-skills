# 蛋仔派对 - 高级功能模块完整教程

## 一、塔防编辑器

### 入口位置

**路径：** 更多 → 规则设置 → 将规则模板设置为【塔防】类型

---

### 塔防规则设置

**位置：** 更多 → 规则设置 → 塔防规则

| 选项 | 说明 |
|------|------|
| 使用官方塔防技能 | 玩家可在局外选择官方技能 |
| 使用官方炮塔 | 玩家可选择官方炮塔艾比 |
| 开局开启第一波进攻 | 开启则自动释放怪物波次 |
| 最后波次结算自动胜利 | 击败最后一轮怪物后自动结算 |
| 星级系统 | 更多 → 规则设置 → 成就 → 星级成就 |
| 图鉴系统 | 更多 → 图鉴编辑 |

---

### 地形制作

**塔防机关位置：** 机关 → 塔防机关分类

| 组件 | 说明 |
|------|------|
| 未解锁塔基 | 解锁后获得塔基 |
| 未解锁-赠送塔 | 解锁后获得塔基和防御塔 |
| 未解锁-金币怪 | 解锁后获得塔基和金币怪 |
| 终点-水晶 | 需要保护的堡垒象征 |
| 终点 | 可配置血量和刷新事件的堡垒 |

---

### 防御塔制作

**炮塔位置：** 生物 → 塔防生物 → 官方炮塔

**自定义防御塔：** 修改生物【特殊生物类型】为<塔防-塔>

**属性配置：**

| 属性 | 说明 |
|------|------|
| 升级事件 | 升级时触发的事件 |
| 升级消耗金币 | 升级所需金币 |
| 升级目标 | 升级后变为的目标预设 |
| 删除返还金币 | 删除时返还的金币数量 |
| 星级 | 设置星级 |
| 删除事件 | 删除时触发的事件 |

**攻击技能：** 使用生物技能里的【攻击技能】作为普攻手段

---

### 生物生成器

**位置：** 机关 → 逻辑分类 → 生物生成器

| 功能 | 说明 |
|------|------|
| 添加生物预设 | 一个生成器可添加多个生物预设 |
| 蛋码路径 | 标记怪物行进方向 |
| 路点标记 | 设置行进路径点 |
| 触发方式 | 设置触发方式 |
| 触发事件 | 触发时发送的事件 |
| 触发延迟时间 | 触发后的延迟 |
| 重复创建次数 | 重复创建的波次数量 |
| 间隔 | 每波之间的间隔 |

---

### 怪物波次配置

**位置：** 更多 → 规则设置 → 塔防规则

| 属性 | 说明 |
|------|------|
| 使波次开启的事件 | 收到事件后生成怪物 |
| 波次开启时发出事件 | 用于计时推进下一波次 |
| 全部击败发出事件 | 自定义胜利结算逻辑 |
| 波次延迟启动时间 | 与生成器延迟叠加 |

---

## 二、任务系统

### 入口位置

**路径：** 更多 → 任务

---

### 创建任务

1. 点击【更多】 → 【任务】
2. 点击左侧"+"号创建新任务
3. 进入任务编辑界面

---

### 任务属性

**位置：** 右侧属性面板

| 属性 | 说明 |
|------|------|
| 任务名 | 在任务看板和追踪面板显示 |
| 任务描述 | 任务看板中展示 |
| 任务标签 | 区分任务类别，有染色信息 |
| 管理标签 | 可修改排序、名称和颜色 |
| 前置条件 | 支持完成任务/接收事件/指定阵营三种类型 |
| 奖励 | 完成后获得 |
| 支持从任务看板接取 | 开关 |
| 失败后可重复尝试任务 | 开关 |
| 完成后自动提交任务 | 开关 |
| 支持快速提交 | 开关 |
| 接取任务事件 | 接收后触发的事件 |
| 接收事件后失败 | 接收事件后任务失败 |
| 展示任务开始横幅 | 开关 |
| 展示任务结束横幅 | 开关 |

---

### 步骤编辑

**路径：** 选中步骤后在右侧编辑

| 属性 | 说明 |
|------|------|
| 顺序 | 决定编辑时的排序 |
| 到达步骤发送事件 | 玩家到达步骤时触发 |
| 指引点 | 屏幕标记当前位置 |
| 指引点类型 | 坐标类/组件类/预设类 |
| 指引点关闭事件 | 收到后取消显示 |
| 判定列表 | 所有可达成判定 |
| 展示目标进度 | 追踪面板展示实时进度 |
| 步骤名 | 追踪面板显示的步骤名称 |

---

### 判定编辑

| 属性 | 说明 |
|------|------|
| 判定类型 | 支持复合条件 |
| 判定完成进度 | 达到进度时判定完成 |
| 判定达成效果 | 跳转步骤/完成/失败任务 |
| 目标完成描述 | 跳转时展示 |
| 结局描述 | 任务完成时横幅展示 |
| 完成发送事件 | 完成时触发全局事件 |
| 开启其他任务 | 完成时解锁其他任务 |

---

## 四、剧情对话系统

### 入口位置

**路径：** 更多 → 剧情对话

**创建：** 点击左侧"+"号

---

### 主要按钮位置

| 按钮 | 位置 | 功能 |
|------|------|------|
| 齿轮按钮 | 左下角 | 打开剧情对话属性设置 |
| 增加选项 | 单条对话编辑界面 | 添加对话选项 |
| 编辑互动内容 | 互动编辑界面 | 进入互动内容编辑 |
| 确认 | 剧情对话选择界面 | 确认选择对话 |
| 退出按钮 | 右上角 | 退出对话 |

---

### 属性编辑入口

| 属性 | 位置 |
|------|------|
| 整体属性 | 左下角齿轮按钮 → 右侧属性面板 |
| 单条对话属性 | 选中单条对话后 → 右侧编辑 |
| 对话选项属性 | 点击【增加选项】后编辑 |

---

### 剧情整体属性

- 剧情名称
- 可由事件触发（配置事件名）
- 可结束时触发事件

---

### 单条对话属性

| 属性 | 说明 |
|------|------|
| 顺序 | 影响对话顺序 |
| 说话者类型 | 生物、玩家或自定义 |
| 说话内容 | 最多50字符 |
| 对话样式 | 设置样式 |
| 蒙版透明度 | 调整透明度 |
| 显示方式 | 一次性或逐字 |
| 播放声音 | 无、音频或AI配音（需10000工匠点） |
| 播放音量 | 调整音量 |
| 对话期间可退出 | 开关 |

---

### 对话选项

| 选项 | 说明 |
|------|------|
| 选项内容 | 最多15字符 |
| 跳转方式 | 终止对话、跳转至其它剧情、本对话指定序号 |

---

### 三种触发方式

1. **组件互动触发：** 勾选组件"开启互动" → 添加互动 → 编辑互动内容 → 选择剧情对话
2. **事件触发：** 设置"可由事件触发"属性 → 通过事件触发区域等触发
3. **蛋码触发：** 使用"开启对话"积木通过事件触发

---

## 五、存档与数据系统

---

### 存档类型

| 类型 | 说明 | 限制 |

| 持有物品 | 保存物品预设和数量 | - |

| 任务 | 记录任务进度 | - |
| 金币 | 游戏内金币数量 | - |

**自定义存档：** 自定义类型存档最多可存储100个条目
