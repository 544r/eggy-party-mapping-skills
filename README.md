# 蛋仔派对地图制作.skills

Claude Code 的蛋仔派对地图制作助手技能。

> 蛋仔派对是网易出品的一款热门手机游戏。本技能帮助用户在移动端工坊编辑器中学习地图制作。

## 为什么做这个

国内某大学开设了一门课程，要求学生制作蛋仔派对地图。然而移动端编辑器的操作方式与PC端不同，官方手册查询不便。为了帮助课程学习者，我们制作了这个 AI 辅助技能，可以：

- 提供手把手的地图制作教程
- 标注每个功能在菜单中的具体位置
- 提供所有蛋码积木的详细参数说明

## 安装方法

```bash
# 克隆到全局 skills 目录
git clone <仓库地址> ~/.claude/skills/蛋仔派对地图制作.skills

# 或手动复制整个文件夹到全局 skills 目录
cp -r 蛋仔派对地图制作.skills ~/.claude/skills/
```

安装后，在任何项目目录下与 Claude Code 提起蛋仔派对相关内容，技能都会自动激活。

## 功能概览

- **核心教程**：编辑器入门、手把手教学指南、常用操作速查
- **组件编辑**：生物组件、地形组件、碰撞预览、地图换色
- **蛋码系统**：400+动作积木、条件积木、事件积木、控制积木
- **高级功能**：UI编辑器、剧情动画、关卡管理、成就系统、存档系统、排行榜等

## 技能激活关键词

蛋仔派对、蛋仔工坊、做地图、地图制作、地图编辑、Eggy Party、eggy、派对地图、以及任何与蛋仔派对地图编辑器相关的问题。
