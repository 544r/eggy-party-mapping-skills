# 蛋仔派对快速参考卡片

快速查询蛋仔派对移动端地图编辑器常用操作。

---

## 界面快捷操作

| 操作 | 方法 |
|------|------|
| 新建地图 | 工坊 → 新建地图 |
| 打开设置 | 点击顶部设置按钮 |
| 试玩地图 | 点击试玩按钮（有普通/沙盒模式） |
| 发布地图 | 设置 → 发布地图 → 填写信息 |
| 切换视角 | 右侧视角旋转工具 |
| 开启精准编辑 | 设置 → 精准编辑 |

---

## 组件操作

| 操作 | 方法 |
|------|------|
| 放置组件 | 从组件库拖拽到场景 |
| 选择组件 | 点击组件 |
| 多选 | 点击多选按钮 → 框选 |
| 移动 | 使用移动工具沿轴拖拽 |
| 旋转 | 使用旋转工具拖拽旋转弧 |
| 缩放 | 使用缩放工具拖拽 |
| 复制 | 拖拽组件出现预览后释放 |
| 删除 | 选中组件 → 删除按钮 |
| 镜像 | 选中 → 镜像按钮 |
| 组合 | 多选 → 组合按钮 |
| 绑定 | 选中子组件 → 拖到父组件上 |

---

## 组件限制

| 组件类型 | 数量限制 |
|----------|----------|
| 地形组件 | 最多10个 |
| 生物组件 | 最多100个（动态创建不限） |
| 生物缩放范围 | 0.1~10倍 |
| 地形最大尺寸 | 254×254 |
| 单次复制上限 | 99个 |

## 蛋码ECA结构

- **E (Event)** - 事件：触发器的启动条件
- **C (Condition)** - 条件：需要满足的要求
- **A (Action)** - 动作：触发器的输出结果

---

## 技能构成

- **触发方式**：冷却时间、充能
- **表现效果**：动画、特效、音效
- **实际伤害**：目标筛选、附加效果

**帧数换算**：1秒 = 30帧

## 坐标轴颜色

- **Y轴** = 绿色（高度）

## 调试技巧

1. **查看运行日志**：蛋码界面有运行日志，点击可跳转问题积木
2. **添加断点**：长按积木 → 添加断点
3. **沙盒模式**：试玩时暂停，逐步测试
4. **性能监测**：设置 → 性能监测

---

## 快速问答

**Q：生物组件怎么调整大小？**
A：不能用缩放工具，需在高级设置→物理→通用属性中修改

**Q：蛋码报错怎么办？**
A：查看运行日志，点击错误信息跳转到问题积木

**Q：地形缩放不了？**
A：地形原点位于角落，不支持旋转和镜像，只能用缩放工具或高级设置调整尺寸

**Q：如何让角色变身生物？**
A：使用「根据生物修改角色形象」动作积木

**Q：组件被锁定了？**
A：检查是否开启了锁定功能，解锁后即可编辑
