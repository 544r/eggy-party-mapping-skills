## 二、编辑界面核心要素

### 场景编辑界面布局

工坊编辑界面是作图过程中最常见的场景编辑界面，主要包含以下功能区域：

- **设置按钮**：点击打开工坊设置和发布地图
- **组件库**（左侧）：选择组件分类，拖拽到场景
- **组件工具栏**（底部，选中组件后出现）：
  - 移动（支持xyz轴移动）
  - 缩放（整体/等比/自由缩放）
  - 旋转（90°/15°/自由旋转）
  - 镜像
  - 组合
  - 删除
  - 绑定
  - 隐藏
  - 锁定
- **试玩按钮**：测试地图
- **视角旋转工具**：调整相机视角，支持快速切换正交视角
- **更多按钮**：打开各种功能菜单

### 组件库分类

**组件库（左侧区域）：**

| 分类 | 内容 |
|------|------|
| 生物 | 蛋仔生物、人形生物、四足动物生物、鱼形等 |
| 地形 | 结构 → 基础形状 |
| 机关-逻辑 | 分阵营出生点、胜利区域、销毁区域、存档区域、事件触发区域、开关、计数器、计时器、随机器等 |
| 机关-功能机关 | 传送门、公告牌、方形重力场、旋律音乐版、打击音乐版、道具箱等 |
| 组合 | 自定义组合组件 |

## 三、坐标系与坐标轴

坐标系分为**世界坐标系**和**局部坐标系**：
- 世界坐标系以（0,0,0）为原点
- 坐标轴颜色编码：X轴红色、Y轴绿色、Z轴蓝色

## 四、发布与测试流程

1. 点击设置按钮 → 发布地图按钮
2. 完成游戏到达终点后填写发布信息
3. 注意：超过地图占用值上限或地图文件大小时，该地图无法发布上传

**试玩模式：**
- 普通模式：正常游戏测试
- 沙盒模式：能在试玩时暂停，以蛋仔视角进行建造活动

**沙盒模式特殊界面：**
- 运行模式 / 暂停模式切换
- 暂停模式下：菜单栏、切换菜单栏、占用值、快捷工具栏（网格显隐、贴合模式、视野）、准星功能、创建面板、飞行模式切换
