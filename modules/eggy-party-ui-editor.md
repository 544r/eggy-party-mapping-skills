## 二、主界面布局

| 区域 | 位置 | 功能 |
|------|------|------|
| 主菜单 | 左上角 | 保存退出、管理、适配 |
| 左侧工具栏 | 左侧 | 蛋码入口、辅助线、多选、复制等 |
| 底部工具栏 | 底部 | 选中控件后显示，可设置属性 |
| 右侧创建面板 | 右侧 | 按住拖拽到画布创建控件 |
| 画布 | 中间 | 只有画布内的控件才会显示给玩家 |

## 六、通用事件

存在4种互动方式：

| 方式 | 触发条件 |
|------|----------|
| 点击 | 按下和松开都在控件范围内 |
| 按下 | 按下控件瞬间触发 |
| 松开 | 松开控件瞬间触发 |
| 长按 | 达到要求时间后触发 |

**事件参数：**
| 参数 | 说明 |
|------|------|
| 仅向界面发送 | 发送方式 |
| 可以交互 | 交互开关 |
| 点击/按下/松开/长按触发事件 | 触发类型 |
| 长按触发时间 | 时间设置 |
| 使控件显示/隐藏的事件 | 显隐控制 |
| 重置动画事件 | 动画重置 |

## 九、常见问题入口

| 问题 | 入口位置 |
|------|----------|
| 找不到出生点/组件 | 右上角组件库搜索栏 |

| 关闭网格吸附 | 设置 → 编辑设置 |
| 修复碰撞检测 | 组件高级设置 → 基础 → 通用属性 → 关闭「组件性能优化」 |
| 设置终点 | 放置「胜利区域」组件 |
