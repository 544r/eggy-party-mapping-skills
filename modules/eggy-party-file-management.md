# 蛋仔派对移动端 - 文件管理功能完整教程

## 一、功能入口

**路径：** 更多 → 文件管理

**开放条件：** 历史最高工匠点达到10000点后开放

---

## 二、上传流程

### 步骤

1. 从【更多】→【文件管理】入口打开界面
2. 点击【上传图片/音频】按钮拉起资源选择界面

### 文件限制

| 类型 | 限制 |
|------|------|
| 图片 | ≤3MB，≤1920×1080 |
| 音频 | ≤1MB，≤60秒 |

### 上传后

上传成功后可在界面浏览文件

---

## 三、使用位置

### 图片使用位置

| 位置 | 说明 |
|------|------|
| 编辑界面 | 拖拽创建图片控件 |
| 预设编辑中 | 作为图标使用 |

### 音频使用位置

| 位置 | 说明 |
|------|------|
| 剧情对话 | 播放声音 |
| 蛋码 | "播放2D音效"积木 |

---

## 四、支持的文件格式

| 类型 | 格式 |
|------|------|
| 图片 | 常见图片格式（PNG、JPG等） |
| 音频 | 常见音频格式（MP3、WAV等） |
