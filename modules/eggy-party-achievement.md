# 蛋仔派对移动端 - 成就系统完整教程

## 一、功能入口

**路径：** 更多 → 规则设置 → 成就

---

## 二、成就配置选项

| 配置项 | 说明 |
|--------|------|
| 成就名称 | 显示在成就列表和完成条幅中 |
| 成就描述 | 完成时弹出显示 |
| 成就品质 | 可设定为隐藏，达成条件显示为"？？？" |
| 达成条件说明 | 地图详情成就列表中展示 |
| 解锁所需计数 | 进度达到此数值时完成成就 |
| 单局成就 | 开启后进度不能存档，重置 |

---

## 三、奖励配置

### 奖励数量
单个成就最多添加**六种**奖励

### 奖励类型

| 类型 | 说明 |
|------|------|
| 角色属性 | 属性加成 |
| 物品 | 物品奖励 |
| 技能 | 技能奖励 |
| 效果 | 效果奖励 |
| 玩家属性 | 玩家属性加成 |
| 自定义事件 | 触发自定义事件 |
| 游玩奖励 | 其他奖励 |

### 领取方式

可设置是否**自动领取**

---

## 四、蛋码配合

通过蛋码写入成就的完成进度

**相关积木：** 成就相关动作和取值积木（在动作类和取值类积木中）
