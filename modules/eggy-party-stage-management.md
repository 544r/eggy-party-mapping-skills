# 蛋仔派对移动端 - 关卡管理完整教程

## 一、功能入口

**路径：** 更多 → 关卡管理

**限制：** 共建模式下暂不支持使用关卡管理功能，共建者只能参与编辑当前关卡或主关卡

**开放条件：** 工匠点≥1万开放

---

## 二、创建多关卡地图

### 创建方法

1. 在关卡管理中点击下方**新建按钮**
2. 可选择：
   - 模版图
   - 草稿箱中已有的草稿（会复制副本作为子关卡，改动不会影响原本草稿）

**注意：** 不会保留子关卡的预设数据

### 数量限制

| 限制 | 数量 |
|------|------|
| 最多关卡数 | 10个（1个主关卡 + 9个子关卡） |
| 单局内最多关卡转换 | 30次 |

---

## 三、关卡编辑

### 主关卡

- 即玩家进入地图时的第一个关卡
- 可以随时更改主关卡
- 主关卡逻辑不会影响其他关卡

### 关卡主要信息

在关卡管理界面编辑关卡的主要信息

---

## 四、关卡切换逻辑

### 切换条件

| 模式 | 切换方式 |
|------|----------|
| 单人模式 | 玩家可直接切换关卡 |
| 多人房间 | 需发起投票，同意票数过半则所有玩家转移至新关卡 |

### 切换流程

1. 触发关卡切换
2. 进入加载状态
3. 玩家在新关卡的出生点出生

### 触发方式

1. **关卡切换区域：** 组件库 → 机关 → 逻辑分类 → 关卡切换区域
2. **蛋码切换关卡：** 积木「发起切换关卡投票」

---

## 五、全局逻辑与关卡逻辑

### 全局逻辑（所有关卡共享）

| 内容 | 说明 |
|------|------|
| 规则设置 | 游戏规则 |
| 预设 | 组件、技能、效果等 |
| 编辑界面 | UI界面 |
| 蛋码 | 全局触发器 |

### 关卡逻辑（仅当前关卡生效）

| 内容 | 说明 |
|------|------|
| 场景组件 | 当前关卡的组件 |
| 天空背景 | 当前关卡的天空 |
| 色彩滤镜 | 当前关卡的滤镜 |

### 蛋码触发器

| 触发器 | 事件 | 生效范围 |
|--------|------|----------|
| 全局触发器 | 游戏初始化 | 所有关卡 |
| 关卡触发器 | 关卡初始化 | 当前关卡 |

### 生效时序

「游戏初始化」在「关卡初始化」之前

---

## 六、多关卡与其他功能

| 功能 | 说明 |
|------|------|
| UI编辑器 | 画布显示设定可设置是否只在当前关卡生效 |
| 效果编辑器 | 效果预设可设置是否在多关卡中继承，关闭后玩家离开当前关卡将失去该效果 |
| 任务编辑器 | 任务可跨关卡结算，支持配置不同关卡的指引点 |
| 存档功能 | 角色存档和玩家存档在不同关卡间继承 |
| 地图商城 | 玩家购买的商品在不同关卡间继承 |
| 物品系统 | 玩家获得的物品在不同关卡间继承 |

---

## 七、出生点要求

**所有关卡出生点信息必须一致**

---

## 八、共建模式注意

共建模式下：
- 关卡管理功能暂不支持
- 共建者只能参与编辑当前关卡或主关卡
