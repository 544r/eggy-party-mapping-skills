# 蛋仔派对移动端 - 规则设置完整教程

## 一、功能入口

**路径：** 更多 → 规则设置

**说明：** 地图规则管理工具，涉及胜利条件、相机设置、地图界面等全局规则

---

## 二、规则设置包含内容

| 模块 | 说明 |
|------|------|
| 阵营 | 设置阵营关系 |
| 排行榜 | 排行榜配置 |
| 地图存档 | 存档设置 |
| 成就 | 成就配置 |
| 计时规则 | 地图时长设置 |
| 属性规则 | 角色属性配置 |
| 其他规则 | 胜利条件等 |

---

## 三、计时规则

### 位置
更多 → 规则设置 → 计时规则

### 功能
修改地图时长

---

## 四、属性规则

### 位置
更多 → 规则设置 → 属性规则

### 功能
切换到复杂模式后可使用攻击力和护甲等属性

---

## 五、阵营设置

### 位置
更多 → 规则设置 → 阵营

### 功能
设置不同阵营之间的关系

---

## 六、胜利条件

### 设置方式
放置「胜利区域」组件

### 位置
组件库 → 机关 → 逻辑分类 → 胜利区域

---

## 七、复活规则

### 角色无法复活
**入口：** 更多 → 规则设置

**检查项：** 检查复活相关规则是否配置正确

---

## 八、相关组件

| 组件 | 路径 |
|------|------|
| 胜利区域 | 组件库 → 机关 → 逻辑分类 |
| 存档区域 | 组件库 → 机关 → 逻辑分类 |
| 出生点 | 组件库 → 机关 → 逻辑分类 |
| 关卡切换区域 | 组件库 → 机关 → 逻辑分类 |

---

## 九、多关卡与规则设置

在多关卡地图中，规则设置属于**全局逻辑**（所有关卡共享）
