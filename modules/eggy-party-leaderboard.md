# 蛋仔派对移动端 - 排行榜系统完整教程

## 一、功能入口

| 类型 | 路径 |
|------|------|
| 局外排行榜 | 更多 → 规则设置 → 排行榜 |
| 局内排行榜 | 游戏过程中实时显示 |
| 排行榜控件 | 更多 → 编辑界面 → 功能分类 |
| 场景排行榜 | 组件库搜索"排行榜" |

---

## 二、排行榜配置

### 基本限制

| 限制 | 数量 |
|------|------|
| 地图最多排行榜条目 | 10个 |
| 详情页可展示 | 3个 |
| 单个排行榜可显示 | 10~100位玩家 |

**注意：** 修改设定时系统会强制重置该排行榜数据

---

## 三、显示位置

| 位置 | 说明 |
|------|------|
| 地图详情页入口界面 | 显示在地图信息中 |
| 局内游戏界面 | 游戏中实时显示 |
| 排行榜控件 | 放置在UI中 |
| 场景排行榜组件 | 有框/无框版本 |

---

## 四、排行榜控件（UI编辑器中）

| 属性 | 说明 |
|------|------|
| 标题 | 显示标题 |
| 排序方式 | 升序/降序 |
| 积分图标 | 图标选择 |
| 数据来源 | 数据源设置 |
| 高亮显示自己 | 开关 |

---

## 五、场景排行榜组件

**位置：** 组件库搜索"排行榜"

**版本：** 有框版本、无框版本

## 六、排行榜与规则设置

排行榜配置位于「更多 → 规则设置 → 排行榜」中
