# 蛋仔派对移动端 - 剧情动画编辑器完整教程

## 一、入口

**路径：** 更多 → 剧情动画

首次使用时，系统会弹出新建页面，可选择心易模板后创建。

---

## 二、编辑设置（左上角「编辑设置」）

| 选项 | 说明 |
|------|------|
| 显示摄像机实体 | 开启后在编辑中可看到并编辑其他摄像机 |
| 显示选中轨道的轨迹 | 选中组件时可在场景中看到运动轨迹 |
| 拖动关键帧时吸附 | 长按拖动关键帧时自动吸附到邻近关键帧或光标线时间 |
| 拖动光标线时吸附 | 拖动光标线时自动吸附到吸附范围内的关键帧时间 |
| 吸附范围 | 决定吸附是否跨轨道生效 |
| 拖动关键帧后选中 | 关闭后拖动关键帧不自动选中 |
| 清理冗余关键帧 | 清除不影响表现的关键帧 |

---

## 三、动画设置（左上角「动画设置」）

| 选项 | 说明 |
|------|------|
| 剧情动画名称 | 用于在蛋码等处调用的标识 |
| 使动画播放的事件 | 玩家接收到该事件时触发动画 |
| 使动画停止的事件 | 接收后立即打断并退出播放 |
| 时长 | 默认30秒，建议与最后一个关键帧对应 |
| 结束时发送的事件 | 播放完毕后发送全局事件 |
| 动画可跳过 | 开启后右上角出现跳过按钮 |
| 屏蔽编辑界面 | 播放时隐藏编辑器创建的UI元素 |

---

## 四、制作剧情动画步骤

### 1. 进入导演轨道
系统只有一条导演轨道

### 2. 添加摄像机分段
1. 选中导演轨道
2. 调整时间轴到目标时刻
3. 点击「添加分段」
4. 选择摄像机

### 3. 摄像机轨道编辑
1. 左下角「当前视口」切换至目标摄像机
2. 移动时间轴
3. 调整摄像机位置/朝向
4. 预览效果

### 4. 将场景对象加入轨道
1. 选中场景组件
2. 点击「添加为轨道」
3. 移动时间轴到目标时刻
4. 按常规方式编辑
5. 预览效果

### 5. 利用预设创建对象
1. 右侧「场景物体/环境氛围」
2. 选择「创建特效/泛光」
3. 选预设
4. 调整当前帧效果
5. 拖动时间轴继续编辑

### 6. 进入轨道编辑模式
1. 左下角点击「轨道编辑」
2. 所有轨道会平铺展示
3. 可精细调整关键帧

## 六、关键帧操作

| 操作 | 说明 |
|------|------|
| 拖动关键帧 | 长按拖动 → 启用吸附（可在编辑设置里调节吸附范围） |
| 拖动光标线 | 受吸附影响，确保精确对齐 |
| 清理冗余关键帧 | 在编辑设置 → 单独分类 → 清理冗余关键帧中点击 |

---

## 七、播放与结束控制

| 功能 | 设置位置 |
|------|----------|
| 播放事件 | 动画设置中设定 |
| 停止事件 | 对应"使动画停止的事件" |
| 跳过功能 | 开启"动画可跳过"后，右上角出现跳过按钮 |
| 结束时事件 | 播放完毕或被跳过时发送全局事件，可在蛋码中继续后续逻辑 |
