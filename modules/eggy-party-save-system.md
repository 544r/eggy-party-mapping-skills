# 蛋仔派对移动端 - 存档系统完整教程

## 一、功能入口

**路径：** 更多 → 规则设置 → 游玩进度

---

## 二、存档类型

| 类型 | 说明 | 限制 |
|------|------|------|
| 存档点 | 存储角色坐标 | 竞速类规则下强制关闭 |
| 胜利清除存档点/位置存档 | 玩家获胜后清除存档 | - |
| 健康值 | 存储角色当前血量 | 生存类规则下强制关闭 |
| 命数 | 存储角色当前命数 | 生存类规则下强制关闭 |
| 等级 | 保存角色等级和经验 | 需开启【等级规则】 |
| 持有物品 | 保存物品预设类型和堆叠数量 | - |
| 分数 | 存储玩家当前积分 | 积分类规则下强制关闭 |
| 任务 | 记录任务进度和已完成的步骤 | - |
| 金币 | 游戏内金币数量 | - |

---

## 三、自定义存档

**位置：** 游玩进度中的"自定义"标签页

| 说明 | 内容 |
|------|------|
| 类型 | 自定义类型的存档 |
| 支持 | 支持存储工匠自定义的数据 |
| 数量限制 | 最多可存储100个条目 |

---

## 四、试玩存档管理

**位置：** 试玩界面中的存档管理功能

| 功能 | 说明 |
|------|------|
| 试玩存档 | 用于在试玩环节测试存档功能 |
| 预设存档修改 | 直接修改存档数据 |

---

## 五、关联地图存档

在地图发布后可在存档设置中选择**继承其他已发布地图的存档条目**

---

## 六、集体存档设置

**操作位置：** 存档区域属性 → 开启「设为集体存档区域」

---

## 七、存档积木

**位置：** 蛋码编辑器 → 动作类别

相关积木：
- 设置存档积木集合
- 获取/设置自定义进度等

---
